# Well-log-correlation.




PETROPHYSICAL & GEOLOGICAL RESERVOIR EVALUATION – SYNTHETIC WELL DATA
A geoscience–data science project simulating reservoir characterization using synthetic well log data.

---

## Overview
This project demonstrates an end-to-end workflow for petrophysical analysis and reservoir evaluation. It generates synthetic well logs (Gamma Ray and Resistivity), derives key petrophysical properties, identifies reservoir zones, and performs inter-well correlation analysis.

The objective is to integrate fundamental geoscience concepts with data-driven techniques for subsurface interpretation.

---

## Key Features
- Synthetic well log generation (GR & Resistivity)
- Petrophysical property estimation (Vshale, Porosity, Water Saturation)
- Reservoir pay zone identification
- Net Pay, Gross Thickness, and NTG calculation
- Inter-well correlation analysis
- Reservoir similarity heatmap visualization

---

## Workflow
1. Synthetic log generation  
2. Petrophysical modeling  
3. Reservoir zone classification  
4. Property computation  
5. Well-to-well correlation  
6. Similarity heatmap visualization  

---

## Tools & Technologies
- Python  
- NumPy  
- Matplotlib  

---

## Results
- Identifies reservoir intervals across multiple wells  
- Captures lateral heterogeneity in reservoir quality  
- Quantifies inter-well similarity on a normalized scale (0–1)  
- Visualizes reservoir continuity using heatmaps  

---

## Conclusion
This project successfully demonstrates a complete synthetic reservoir characterization workflow combining petrophysics, reservoir modeling, and data analysis. It highlights the application of geoscience principles using computational methods for subsurface evaluation.
